# Mantenimiento Preventivo — App PWA

Aplicación web progresiva (PWA) para el programa de mantenimiento preventivo de flota vehicular.

## Instalar en tu dispositivo

### Paso 1 — Publicar en GitHub Pages (gratis)

1. **Crear repositorio en GitHub**
   - Ve a [github.com](https://github.com) → Inicia sesión
   - Clic en **"New repository"** (botón verde)
   - Nombre: `mantenimiento-app`
   - Visibilidad: **Public**
   - Clic **"Create repository"**

2. **Subir los archivos**
   - En tu nuevo repositorio, clic en **"uploading an existing file"**
   - Arrastra TODOS estos archivos/carpetas:
     - `index.html`
     - `manifest.json`
     - `sw.js`
     - carpeta `icons/` (con todos los iconos .png)
   - Escribe el mensaje: `Primera versión`
   - Clic **"Commit changes"**

3. **Activar GitHub Pages**
   - En tu repositorio, ve a **Settings** (pestaña superior)
   - En el menú izquierdo: **Pages**
   - En "Source" selecciona: **Deploy from a branch**
   - Branch: **main** / Folder: **/ (root)**
   - Clic **Save**
   - Espera 1-2 minutos → Tu URL será:
     `https://TU-USUARIO.github.io/mantenimiento-app/`

---

### Paso 2 — Instalar en iPhone

1. Abre Safari en tu iPhone
2. Entra a tu URL de GitHub Pages
3. Toca el botón **Compartir** (cuadrado con flecha hacia arriba)
4. Desplázate y toca **"Añadir a pantalla de inicio"**
5. Nombre: `Mtto Prev` → toca **Añadir**
6. ¡La app aparece en tu pantalla de inicio!

### Paso 2 — Instalar en Android

1. Abre Chrome en tu Android
2. Entra a tu URL de GitHub Pages
3. Aparecerá un banner automático **"Instalar app"** → tócalo
4. Si no aparece: toca los **3 puntos** (menú) → **"Añadir a pantalla de inicio"**
5. ¡La app aparece como app nativa!
